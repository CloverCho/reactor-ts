get

  • get(): Present | null

  • Inherited from Action.get

    • Defined in core/reactor.ts:504

    Called on an action within a reaction to acquire the action's value. The value for an action is set by a scheduled action event, and is only present for reactions executing at that logical time. When logical time advances, that previously available value is now unavailable. If the action was scheduled with no value, this function returns null.
